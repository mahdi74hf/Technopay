# پنل تکنوپی — Technopay Panel

پنل مدیریت پذیرندگان و تسویه، با دو نقش **پذیرنده (merchant)** و **ادمین (admin)**.
ساخته‌شده با **React + TypeScript + Vite**. رابط کاربری کاملاً راست‌به‌چپ (RTL) و فارسی.

## اجرا

```bash
npm install
npm run dev      # اجرای محیط توسعه روی http://localhost:5173
npm run build    # بیلد پروduction در پوشه‌ی dist
npm run preview  # پیش‌نمایش بیلد
npm run typecheck
```

> نیازمندی: Node.js نسخه ۱۸ یا بالاتر.

## ورود دمو

| نقش | نام کاربری | رمز عبور | کد OTP |
|-----|-----------|----------|--------|
| پذیرنده | `merchant` | `merchant` | `1234` |
| ادمین | `admin` | `admin` | `1234` |

## امکانات

- **داشبورد** — کارت‌های KPI، نمودار روند روزانه (تعاملی)، دونات آنلاین/آفلاین، نمودار شعبه و صندوقدار، و فیلترهای زنده (بازه، شعبه، کانال، صندوقدار).
- **گزارشات** — جدول تراکنش‌ها + جستجو + فیلتر + جزئیات هر تراکنش (drawer) + خروجی اکسل (CSV).
- **در انتظار تسویه** — کارت‌های خلاصه + جدول + خروجی اکسل.
- **پروفایل پذیرنده** — اطلاعات عمومی، حساب و قرارداد، تسویه و تاریخچه تغییرات.
- **مدیریت دسترسی** — افزودن/حذف شعبه و صندوقدار.
- **پنل ادمین** — دید روی همه‌ی پذیرندگان + فیلتر «پذیرنده» در نوار بالا.

## ساختار

```
src/
  main.tsx              نقطه ورود، AppProvider را mount می‌کند
  App.tsx               روتینگ سطح‌بالا (login ⇄ app + ناوبری صفحات)
  store.tsx             وضعیت سراسری + اکشن‌ها + خروجی CSV (React Context)
  theme.ts              توکن‌های رنگ
  types.ts              تایپ‌های دامنه و وضعیت اپ
  lib/
    format.ts           اعداد فارسی، جداکننده، مبلغ، فشرده (میلیون/میلیارد)
    jalali.ts           تبدیل تاریخ میلادی به شمسی
    data.ts             ساخت داده‌ی نمونه (seeded) — اینجا را با API واقعی جایگزین کنید
    selectors.ts        توابع خالص فیلتر/اسکوپ + گزینه‌های فیلتر
  components/
    Login.tsx  Sidebar.tsx  Topbar.tsx  FilterBar.tsx  Drawer.tsx
    charts/   TrendChart, ChannelDonut, BranchBars, CashierBars
    pages/    Dashboard, Reports, Settlements, Profile, Access
```

## اتصال به داده‌ی واقعی

تمام داده از `src/lib/data.ts` (تابع `buildData`) می‌آید و یک شیء از نوع `DataSet` برمی‌گرداند.
برای اتصال به بک‌اند، این تابع را با یک fetch جایگزین کنید که همان شکل `DataSet` را تأمین کند؛
بقیه‌ی اپ بدون تغییر کار می‌کند.

## نکته

نمودارها بدون کتابخانه‌ی جانبی و با SVG سبک پیاده شده‌اند (فقط react و react-dom).
اگر بعداً نمودار پیچیده‌تر خواستید، افزودن `recharts` ساده است.
